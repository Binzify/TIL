- Counting Antibodies (잔디)

  - ```python
    a, b = map(int,input().split())
    c, d = map(int,input().split())
    e, f, g = map(int,input().split())
    
    chainA = a * b
    chainB = c * d
    answer = e*f*g
    print(answer*(chainA+chainB))
    ```
