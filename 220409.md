  - 처음에는 이게 되겠어..? 하고 바로 짜서 집어넣었는데 통과해서 이게 실버라고..? 했다.

  - ```python
    N = int(input())
    line = list(sorted(map(int, input().split())))
    atm = []
    
    for i in line:
        if not atm:
            atm.append(i)
        else:
            atm.append(i+atm[-1])
    print(sum(atm))
    ```
