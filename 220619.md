- 상근날드 (잔디)
  - 세 개의 햄버거와 두 개의 음료수가 주어지면 거기서 min 값을 뽑아 더한 후 -50원을 계산해서 출력하면 된다.
  
  - ```python
    burgers = []
    drinks = []
    
    for _ in range(3):
        burgers.append(int(input()))
    
    for _ in range(2):
        drinks.append(int(input()))
    
    cost = 0
    cost = min(burgers) + min(drinks) - 50
        
    print(cost)
    ```
