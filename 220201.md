### 💁🏼‍♂️가장 기억에 남는 교육 내용

while 은 조건 달성할 때까지 계속 반복하는 것

```python
while True:
    A, B = map(int, input().split())
    if A + B != 0 :
        print(A + B)
    else:
        break
```

map (int,input().split()) << 매번 따로따로 정수화 시켰는데... 그럴필요가 음슴!!

while True로 해두고 끝내고 싶을 땐 break 문 쓰기!!!! return 노노해~

for 문은 정해두는 것!
