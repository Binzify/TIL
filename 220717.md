#### 백준

- FA
  
  - ```python
    num = input()
    while True:
        num = int(num[0]) * len(num)
        num = str(num)
        if len(num) == 1:
            if num[0] == num:
                print('FA')
                break
            else:
                print('NFA')
                break
    ```
  - 주어진 수의 맨 앞자리와 자릿수를 곱해서 최종적으로 한 자리수가 나오면 그 값을 비교해서 같은지 판별하면 된다.
  - int와 str의 구조를 이해함은 물론 while을 사용할 때 if문으로 break를 어떻게 걸어야하는지 다시 한 번 생각해볼 수 있었던 시간이었다.
