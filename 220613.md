#### 백준

- 5의 수난(잔디)

  - 문자열로 받아서 하나씩 뽑아 제곱해서 더해주기

  - ```python
    num = input().rstrip()
    
    sum = 0
    for i in range(5):
        new = int(num[i])
        sum += new**5
        
    print(sum)
    ```
