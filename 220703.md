- 과목선택
  - 주어지는 입력 중 가장 큰 점수를 받아서 max 값을 구하는 것
  
  - 슬라이싱 활용해서 점수를 나누기
  
  - ```python
    subjects = [] 
    for _ in range(6):  
        subjects.append(int(input())) 
    subject1 = sorted(subjects[:4]) 
    subject2 = subjects[4:] 
    print(sum(subject1[1:]) + max(subject2))
    ```
