  - ```python
    plus, minus = map(int,input().split())
    vers = []
    
    x = plus+minus
    y = plus-minus
    z = x//2
    
    if x < 0 or y < 0 or x%2:
        print(-1)
    else:
        vers.append(z)
        vers.append(plus-z)
        print(*vers)
    ```
