#### 백준

-   수학은 체육과목 입니다 2
  - 1은 +8이 될때마다 걸리게 된다. 그러므로 나머지를 했을 때 나오는 수로 손가락이 무엇인지 확인할 수 있다.
  - ```python
    num = int(input())
    
    
    finger = num % 8
    
    if finger == 1:
        print(1)
    elif finger == 2 or finger == 0:
        print(2)
    elif finger == 3 or finger == 7:
        print(3)
    elif finger == 4 or finger == 6:
        print(4)
    else:
        print(5)
    ```
