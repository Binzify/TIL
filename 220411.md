#### 백준

- 피보나치 수 (DP)

  - 메모이제이션 활용하여 문제 해결

  ```python
  N = int(input())
  memoization = [0, 1]
  def fibo(n):
      global memoization
      if n >= 2 and len(memoization) <= n:
          memoization.append(fibo(n-1)+fibo(n-2))
      return memoization[n]
  
  print(fibo(N))
  ```

- 피보나치 수 2

  - 위와 동일한 코드로 문제 해결함
  - 메모이제이션에 대한 이해도를 높이고 DP와 조금 더 .... 친해진 느낌이랄까...
