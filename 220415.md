  - 같이 프로젝트를 진행하면서 부족한 부분들을 채워나가는 느낌이었다.

  - 생각보다 협업은 어렵지 않았다. Git Branch 과정을 거치지 않고 하나의 master 브랜치로 작업을 해서 수월했던 것 같았다.
